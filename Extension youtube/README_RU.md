**YouTuDo** — это расширение для браузера, работающее в связке с мощным локальным сервером. Оно позволяет извлекать и скачивать видео с YouTube (включая **4K**) с любой звуковой дорожкой, ловко обходя защиты и ограничения скорости.

## ⚠️ Системные требования (ПК)

Для оптимальной работы YouTuDo необходим небольшой локальный "движок". Убедитесь, что у вас установлено следующее:

| Программа | Описание и ссылка для скачивания |
| :--- | :--- |
| 🐍 **Python 3** | Необходим для запуска локального API-сервера.<br>🔗 **[Скачать Python](https://www.python.org/downloads/windows/)** |
| 🟩 **Node.js** | **Критически важно.** Расшифровывает защиту от ботов YouTube для скачивания на максимальной скорости. *(Должен быть установлен в папку по умолчанию: `C:\Program Files\nodejs`)*.<br>🔗 **[Скачать Node.js (LTS-версия)](https://nodejs.org/)** |
| 🎬 **FFmpeg** | **Обязательно.** Инструмент для объединения 4K-видео с выбранной аудиодорожкой. *(Исполняемый файл должен быть добавлен в `PATH` Windows)*.<br>🔗 **[Скачать FFmpeg (сборки для Windows)](https://www.gyan.dev/ffmpeg/builds/)** или через [GitHub](https://github.com/BtbN/FFmpeg-Builds/releases) |
| 🌐 **Браузер** | Браузер на базе Chromium (Google Chrome, Microsoft Edge, Brave и т.д.). |

## ⚙️ Руководство по установке

### 1️⃣ Настройка локального сервера (Backend)
Сервер выполняет всю тяжелую работу в фоновом режиме (извлечение и объединение).

1. Откройте терминал (Командную строку или PowerShell) в папке `youtube-api`.
2. Установите необходимые зависимости Python, введя:
   ```cmd
   pip install fastapi uvicorn yt-dlp
   ```
3. Запустите сервер:
   ```cmd
   python -m uvicorn main:app --reload
   ```
   > 💡 **Совет:** Оставьте этот терминал открытым. Сервер будет автоматически проверять наличие обновлений `yt-dlp` (основа загрузчика) в фоновом режиме при каждом запуске!

### 2️⃣ Установка расширения (Frontend)
1. Откройте браузер и перейдите на страницу управления расширениями (введите `chrome://extensions/` или `edge://extensions/` в адресной строке).
2. Включите **«Режим разработчика»** (обычно это переключатель в правом верхнем углу).
3. Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked).
4. Выберите корневую папку расширения (главную папку, содержащую файл `manifest.json`).

## 🚀 Повседневное использование

1. **Подготовка**: Убедитесь, что ваш Python-сервер (`main.py`) запущен.
2. **Навигация**: Перейдите на страницу нужного видео на YouTube.
3. **Выбор**: Нажмите на значок расширения YouTuDo на панели инструментов. Откроется интерфейс, где загрузятся все доступные форматы (включая скрытые дубляжи).
4. **Скачивание**: Выберите желаемое разрешение или аудиодорожку и начните скачивание одним щелчком мыши!

> ⚠️ **Важное замечание:** Не закрывайте окно терминала во время скачивания видео, иначе операция будет отменена.
